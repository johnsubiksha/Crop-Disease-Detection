# Crop Disease Detection using ResNet50

## Overview
This project uses Transfer Learning with ResNet50 to detect crop diseases from leaf images.

## Features
- Crop disease classification
- Transfer learning using ResNet50
- Image preprocessing and augmentation
- Disease prediction from uploaded images

## Technologies Used
- Python
- TensorFlow
- ResNet50
- OpenCV
- NumPy
